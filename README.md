# Studio Verde - Архитектурная и интерьерная студия

Веб-сайт для архитектурной и интерьерной студии "Studio Verde", созданный с использованием Node.js, Express, Sequelize и SQLite.

## Описание

Главная страница веб-сайта архитектурной и интерьерной студии «Studio Verde» выполнена в минималистичном стиле с акцентом на визуальный контент и экологичную эстетику. Основная задача страницы — произвести первое впечатление, представить студию, продемонстрировать ключевые проекты и побудить пользователя к дальнейшему взаимодействию с сайтом.

## Технологии

- Node.js
- Express
- Sequelize (ORM)
- SQLite
- EJS (шаблонизатор)
- HTML/CSS/JavaScript
- Responsive Design

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Синхронизируйте базу данных:
```bash
npm run sync-db
```

3. Запустите сервер:
```bash
npm start
```

Приложение будет доступно по адресу: http://localhost:3000
